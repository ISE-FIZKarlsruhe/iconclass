Demo: Iconclass Multimodal Search

This Demo Interface is based on the system described in Multimodal Search on Iconclass using Vision-Language Pre-Trained Models (arXiv).

By using the search bar below, a visual similarity search will be performed in order to find relevant images and Iconclass notations in the Arkyves database. The system will convert your input into multimodal embeddings by using OpenAI's CLIP.
